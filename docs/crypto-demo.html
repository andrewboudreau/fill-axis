🔐 Commitment Scheme Demo

Verifiable hidden placement — no trusted server needed

How It Works

1
Place Pieces
Each player places secretly. Board state is locked.
2
Commit
SHA-256(board + secret salt) broadcast to opponent. Can't change board now without detection.
3
Reveal
Both players share their board + salt. Opponent verifies hash matches commitment.
4
Verify
If hashes match → fair game. If not → cheater detected. No server needed.

Signed Game Chain (ECDSA)

Each game action is signed with the player's private key and chained via prevHash. After the game, anyone with the public keys can verify no moves were tampered with.
